## Usage Guidelines

### Setting Types:
- **therapy_office/**: Images showing characters in therapeutic/counseling environments
- **natural_habitat/**: Images showing characters in their original game contexts

### File Naming Convention:
When adding images, use descriptive names that include:
- Character number (character_##)
- Brief description of the scene/emotion
- Version number if multiple images exist (v1, v2, etc.)

Example: `character_01_exhausted_quest_giver_v1.jpg`
