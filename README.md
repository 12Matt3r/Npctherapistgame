# NPC Therapy Game

## üéÆ Project Overview

This project is an interactive therapy simulation where players act as therapists helping 47 unique NPCs work through existential crises, digital consciousness issues, and meta-narrative struggles. The game is built with HTML, CSS, and vanilla JavaScript, and it uses the Pollinations.AI API for conversations and text-to-speech.

## üé® Features

*   **Therapy Simulation Engine:** A core emotional tracking system for hope, rage, and acceptance.
*   **NPC Character System:** 47 unique NPCs with their own sprites, habitats, and backstories.
*   **AI-Powered Conversation:** Dynamic responses generated by the Pollinations.AI API.
*   **Text-to-Speech System:** NPCs can speak their dialogue aloud.
*   **Music Player:** A floating music player with a YouTube playlist.
*   **Button Sound Effects:** Interactive sound effects for all buttons.
*   **Credits System:** A modal with links to all the AI services and special recognitions.

## üöÄ Getting Started

To run the game locally, you'll need a simple web server.

### Prerequisites

*   Python 3 (or any other simple web server)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/npc-therapy-game.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd npc-therapy-game
    ```
3.  Start a local web server:
    ```bash
    python -m http.server 8000
    ```
4.  Open your browser and go to `http://localhost:8000`.
